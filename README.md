# Userscripts

A collection of userscripts for Tampermonkey.

## Scripts

| Name | Description | Install |
|------|-------------|---------|
| YouTube Ad Auto Skipper | Automatically skips YouTube ads (skip button + seek fallback) and dismisses ad blocker warning dialogs. | [Install](https://github.com/sammrai/userscripts/raw/main/youtube_ad_skipper/youtube_ad_skipper.user.js) |
| ChatGPT Send Message on Cmd+Enter | Sends messages with Cmd+Enter (or Ctrl+Enter) instead of Enter alone. Useful for Safari users typing in Japanese. | [Install](https://github.com/sammrai/userscripts/raw/main/chatgpt_send_message_on_cmd_enter/chatgpt_send_message_on_cmd_enter.user.js) |
| X Tweets → Markdown | Collects visible tweets and copies them as Markdown to the clipboard. | [Install](https://github.com/sammrai/userscripts/raw/main/x_tweets_to_markdown/x_tweets_to_markdown.user.js) |
| YouTube IINA Button | Adds an IINA button to YouTube video pages to open the video in IINA at the current playback position. Requires [this Shortcut](https://www.icloud.com/shortcuts/879c00db8c4b4f0f99bf694458d006ec). | [Install](https://github.com/sammrai/userscripts/raw/main/youtube_iina_button/youtube_iina_button.user.js) |
| Etherscan JST | Converts UTC timestamps on Etherscan to Japan Standard Time (JST). | [Install](https://github.com/sammrai/userscripts/raw/main/etherscan_jst/etherscan_jst.user.js) |
| Telegram Hide Stickers | Hides messages containing stickers in Telegram Web. | [Install](https://github.com/sammrai/userscripts/raw/main/telegram_hide_stickers/telegram_hide_stickers.user.js) |
| ふるさと納税 自治体別合計サイドパネル | ふるさと納税カートページに自治体別の合計・商品内訳・カテゴリ別重量の去年比をサイドパネルで表示。 | [Install](https://github.com/sammrai/userscripts/raw/main/furusato_summary_panel/furusato_summary_panel.user.js) |
