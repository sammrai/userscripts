# Userscripts

A collection of userscripts for Tampermonkey.

## Scripts

| Name | Description | Install |
|------|-------------|---------|
| YouTube Ad Auto Skipper | Skips YouTube ads automatically via skip button click and seek fallback. Also dismisses ad blocker warning dialogs. | [Install](https://github.com/sammrai/userscripts/raw/main/youtube_ad_skipper/youtube_ad_skipper.user.js) |
| YouTube Open in IINA | Adds an "IINA" button to YouTube that opens the current video at the current playback position in IINA. Requires [this Shortcut](https://www.icloud.com/shortcuts/879c00db8c4b4f0f99bf694458d006ec). | [Install](https://github.com/sammrai/userscripts/raw/main/youtube_iina_button/youtube_iina_button.user.js) |
| ChatGPT Cmd+Enter to Send | Changes the send trigger on ChatGPT from Enter to Cmd+Enter (or Ctrl+Enter), preventing accidental submission when confirming Japanese input. | [Install](https://github.com/sammrai/userscripts/raw/main/chatgpt_send_message_on_cmd_enter/chatgpt_send_message_on_cmd_enter.user.js) |
| X (Twitter) Tweets to Markdown | Collects tweets visible on the page and copies them as formatted Markdown (with author, date, and link) to the clipboard. | [Install](https://github.com/sammrai/userscripts/raw/main/x_tweets_to_markdown/x_tweets_to_markdown.user.js) |
| Etherscan Timestamp to JST | Converts UTC timestamps on Etherscan transaction pages to Japan Standard Time (JST). | [Install](https://github.com/sammrai/userscripts/raw/main/etherscan_jst/etherscan_jst.user.js) |
| Telegram Web Hide Sticker Messages | Hides messages that contain stickers in Telegram Web. | [Install](https://github.com/sammrai/userscripts/raw/main/telegram_hide_stickers/telegram_hide_stickers.user.js) |
| ふるさとチョイス カート集計パネル | ふるさとチョイスのカートページに、自治体別の金額・商品一覧・カテゴリ別重量（去年比付き）をサイドパネルで表示。 | [Install](https://github.com/sammrai/userscripts/raw/main/furusato_summary_panel/furusato_summary_panel.user.js) |
